# Portfolio Website

This is my personal portfolio website built with React and styled with custom CSS. It showcases my projects, skills, and provides a way to contact me.

## Features

- Responsive design ensuring the website is accessible on all devices.
- React Router for smooth navigation without page reloads.
- Dynamic project sections created with reusable React components.

## Installation

To run this project locally, you'll need to have Node.js installed. After cloning the repository, install the dependencies:
